I noticed that the dataset values were already normalized in the range [0, 1].  I apply greyscaling to reduce simplify the input while retaining as much information as possible. This improves the validation accuracy by about 1-2%.  To normalize the input further, I apply a transformation that scales the minimal value down to 0 and the max to 1 for every image in the dataset. The reason for this is to enhance the contrast so that the absolute gradient between the digit and the background is less varied amongst different images. This increases the validation accuracy by another 1-2%.  This results in a validation accuracy of 0.806.  (length: 639)

I started of with creating a 2 convolutional layers with 64 filters to put in front of the previous dense network. I use kernel_size = (3,3) with padding = 'same', with stride 1 as the digits occupy a low amount of pixels. I applied a pooling layer to select only on the most important features, and a batch normalization layer to normalize the activations. This resulted in a validation accuracy of 0.905.  Adding an additional set of 2 convolutional layers, 1 pooling and a batch normalization layer resulted in a validation accuracy of 0.932.  I changed the structure of the dense layer section to have dense 512 with relu to our output dense 10 with softmax, obtaining 0.940 validation accuracy. At this point the training accuracy was 0.977, indicating that the network is overfitting.  To reduce overfitting I added dropout layers which improved the results to 0.951. L2-regression negatively affected performance for various values. Finally I optimized the hyperparameters by looping over various values for dropout, learning rate and batch size to come to a performance of 0.958.  (length: 1089)

    model.add(layers.BatchNormalization())  
    model.add(layers.MaxPooling2D(pool_size=(2,2)))
    model.add(layers.Dropout(0.45))
    
    model.add(layers.Conv2D(filters=128, kernel_size = (3,3), padding='same', activation="relu"))
    model.add(layers.BatchNormalization())
    model.add(layers.Conv2D(filters=128, kernel_size = (3,3), padding='same', activation="relu"))

def plot_3_3():
    model = load_model_from_file("", "model_2_2", extension='.h5')
    
    # Get the image tensor and preprocess
    img_tensor = X_test[0]
    img_tensor = np.expand_dims(img_tensor, axis=0) 
    img_tensor_pre = rgb2gray(img_tensor)
    
    # Get the model prediction and the last conv layer
    prediction = model.predict(img_tensor_pre)
    argmax = np.argmax(prediction[0])
    output = model.output[:, argmax]
    last_conv_layer = model.layers[15]
    
    # Get the convolutional layer output
    grads = B.gradients(output, last_conv_layer.output)[0]
    pooled_grads = B.mean(grads, axis=(0, 1, 2))
    iterate = B.function([model.input], [pooled_grads, last_conv_layer.output[0]])
    pooled_grads_value, conv_layer_output_value = iterate([img_tensor_pre])
    for i in range(127):
        conv_layer_output_value[:, :, i] *= pooled_grads_value[i]
    
    # Get the heatmap
    small_heatmap = np.mean(conv_layer_output_value, axis=-1)
    small_heatmap = np.maximum(small_heatmap, 0)
    small_heatmap /= np.max(small_heatmap)

    # Resize the heatmap
    W = img_tensor.shape[1]
    H = img_tensor.shape[2]
    res = np.linspace(0, (small_heatmap.shape[0]*H-1)/H, H)
    heatmap = np.zeros((W, H, 1))
    for w in range(W):
        for h in range(H):
            heatmap[h, w, 0] = small_heatmap[int(res[h]), int(res[w])]
    
    superimposed_img = np.concatenate((heatmap, np.squeeze(img_tensor, axis=0)[:,:,1:]), axis=2)

    # Plot
    plt.imshow(superimposed_img)
    plt.axis('off')
    plt.title('Activations')
