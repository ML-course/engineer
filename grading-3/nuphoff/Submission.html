I've created a CNN with 3 conv layers with increasing amount of filters 32 to 64 to 128, 
the intuition behind this was that every subsequent conv layer can combine multiple features from the previous one 
which results in more posibilities.
I've added Maxpooling (to create an abstracted form of the input), batchnormalization (to improve the learning speed) 
and dropout (to reduce overfitting) to these conv layers.
After the conv layers we flatten the input and feed it trough a dense layer and finally using softmax with categorical crossentropy like before.
I've added l2 regularization to the conv and dense layer to prevent overfitting (much lower training loss than validation loss).
The model performs fairly well because the validation it's not under or overfitting within 15 epochs.
